# 物流管理系统(Logistics Management System)

## 项目介绍
本项目是基于**SpringBoot+Vue3**的全栈物流管理系统，旨在模拟真实的快递流转业务。
作为个人的本科毕业设计。项目涵盖了从用户下单，后台调度到快递员配送的完整闭环流程

## 🛠 技术栈
### 后端 (Backend)
- **核心框架**: Spring Boot 3.x
- **持久层**: MyBatis-Plus (简化 CRUD 操作)
- **数据库**: MySQL 8.0
- **工具类**: Lombok, Jackson

### 前端 (Frontend)
- **核心框架**: Vue 3 (Composition API)
- **UI 组件库**: Element Plus
- **网络请求**: Axios
- **状态管理**: Pinia

---

## 核心功能 (Roadmap)
- [ ] **权限管理**: 基于角色的访问控制 (管理员、用户、快递员)。
- [ ] **订单管理**: 实现订单的创建、查询、修改及逻辑删除。
- [ ] **物流轨迹**: 记录快递在各站点的流转时间线。
- [ ] **看板分析**: 后台首页展示快递总量、待配送量等可视化统计。
